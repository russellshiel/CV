About Me

Hey! I'm Russell. A hardworking and professional website designer with extensive experience in the understanding of web content and what people look for in a website. My skill set includes HTML, XHTML, and CSS etc. I have extensive search engine optimisation experience and a solid understanding of paid search using Google AdWords.

I specialise in improving and maintaining keyword density, plus I have an eye for making websites look visually appealing and user-friendly to improve the customer experience.

I have also recently learnt how to create responsive websites using the Jekyll program and Markdown.

Web Designer at Shiftec Ltd, Leamington Spa - November 2020 – April 2021 – Six-month contract

Shiftec is a Tier 1 supplier and engineered products and control electronics business with a heritage dating back to 1920. As the Web Designer, my core activities included:

  • Work on a project called InTiFi to help implementation of the new Aquila System. The Aquila system is a suite of software modules that can be integrated into different industries and plant processes to gain an increased level of automation, efficiency, accuracy and traceability.
  • These modules were web based to give maximum accessibility to the staff within the plant. This will allow users to use the system from any number of different devices including, workstations, laptops, tables and smart phones.
  • The web-based modules where split into two separate websites. One will be for setting up, configuring and monitoring the Aquilla system. The other will be for operator use, providing the required functionality in an easy to use front end. The proposed frameworks for this will be ASP.net using HTML/CSS front end design using Bootstrap 4
